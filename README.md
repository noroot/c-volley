# C-Volley

Volley game inspired by Blobby Volley, written in C using RayLib.

![./c-volley-logo.png](./c-volley-logo.png)

## Screenshots

![./screenshot-0.jpeg](./screenshot-0.jpeg)

## Build from source
- Install or compile raylib from source
- run `make run` to compile and run the game
